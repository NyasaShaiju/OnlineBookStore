Online Bookstore

A Java-based web application that allows users to browse and purchase books online. The system supports user authentication and basic bookstore management features.

Features

User registration and login system

Role-based access (Admin & Customer)

View and manage books

Add, update, and delete book details (Admin)

Database-driven application using MySQL

Tech Stack

Backend: Java (Servlets, JSP)

Frontend: HTML, CSS

Database: MySQL

Server: Apache Tomcat

Build Tool: Maven

Version Control: GitHub
